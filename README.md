# QA AutoTest Pet-project

Pet-проект по автоматизации тестирования на Java. Репозиторий содержит несколько модулей, демонстрирующих навыки работы с UI-, API- и DB-тестами.

## 🛠️ Используемые технологии

- **Java 17+** — основной язык
- **Selenium WebDriver** — UI-автоматизация
- **JUnit 5** — запуск тестов, параметризация, ассерты
- **RestAssured / OkHttp** — тестирование REST API
- **Hibernate / JDBC** — работа с базами данных
- **Jackson** — работа с JSON

## 📁 Краткая структура

| Модуль | Что делает |
| :--- | :--- |
| `addressbook_web_tests26` | UI-тесты веб-приложения, проверка и подготовка данных в БД, тесты REST API (OkHttp)|
| `rest-api-tests` | Тесты REST API (RestAssured)|
| `mantis` | UI-тесты веб-приложения, работа с локальным почтовым сервером |
